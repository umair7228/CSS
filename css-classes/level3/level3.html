Relative Units in CSS

relative units are not fixed. Their values are "relative" to another value.

1. Percentage (%)

It is often used to define a size as relative to an element's parent object.

2. em Unit

Relative to the font-size of the element (2em means 2 times the size of the current font)

3. rem Unit

Relative to the font-size of the root element (2rem means 2 times the size of the root font)

Position Property in CSS

The position CSS property sets how an element is positioned in a document.

1. Static Property

static - default position (The top, right, bottom, left, and z-index properties have no effect)

2. Relative Property

element is relative to itself. (The top, right, bottom, left, and z-index will work)

3. Absolute Property

element is relative to its closest positioned ancestor. (The top, right, bottom, left, and z-index will work)

4. Fixed Property

positioned relative to browser. (removed from flow)

5. Sticky Property

positioned based on user's scroll position

To Add Background Image

The background-image property sets one or more background images for an element.

1. Cover Property

Resize the background image to cover the entire container, even if it has to stretch the image or cut a little bit off one of the edges

2. Contain Property

Resize the background image to make sure the image is fully visible
